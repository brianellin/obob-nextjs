A big part of OBOB is learning each author's name and being able to pronounce it correctly during a live battle.

You can now tap an underlined author name to quickly view the pronunciation!

This feature helps students prepare for the pronunciation portion of OBOB battles, where correctly saying an author's name can make the difference between winning and losing a point.
